# Welcome to My Blog

This site is generated from Markdown into static HTML.

## Why this approach

- Fast pages
- No client-side framework
- Easy deployment on GitHub Pages

## Syntax support

The parser supports **bold**, *italic*, and `inline code`.

> Simple tools and clear structure usually age better.
